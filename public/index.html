Ajuda e Documentação

Gerenciar Produtos

  • Adicionar: Clique em "Adicionar Produto" ou use Ctrl+N
  • Editar: Clique no ícone de lápis no produto
  • Remover: Clique no ícone de lixeira
  • Filtrar: Use os filtros acima da lista

Sistema de Descontos

  • Categoria: Aplica desconto apenas à categoria selecionada
  • Cupom: Aplica desconto a todos os produtos
  • Cumulativo: Descontos são aplicados em sequência
  • Preview: Use "Visualizar" para ver o efeito

Atalhos do Teclado

  • Ctrl + N - Adicionar produto
  • Ctrl + S - Salvar (no modal)
  • ESC - Fechar modal/painel
  • F1 - Abrir ajuda
  • / - Focar na busca
